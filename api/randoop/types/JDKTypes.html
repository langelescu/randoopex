Package randoop.types

Class JDKTypes

java.lang.Object
randoop.types.JDKTypes
public class JDKTypes extends Object
Defines type constants for classes in the JDK Collections.

These types should be maintained as new JDK versions are released. They have been updated for JDK 8.

    • getImplementingTypeForCollection

      public static GenericClassType getImplementingTypeForCollection(ParameterizedType type)
      Returns an arbitrary (but fixed) concrete collection type for each Collections type (e.g., returns ArrayList for List). If a type is already concrete, then returns that type.
      Parameters:
      type - the (abstract) Collections type
      Returns:
      a concrete Collection type implementing the given type
