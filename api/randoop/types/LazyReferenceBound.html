Class LazyReferenceBound

java.lang.Object
randoop.types.ParameterBound
randoop.types.ReferenceBound
randoop.types.LazyReferenceBound
class LazyReferenceBound extends ReferenceBound
A lazy representation of a type bound in which a type variable occurs. Similar in purpose to LazyParameterBound, but this class uses ReferenceType as the bound instead of java.lang.reflect.Type. Also, prevents access to recursive type bounds, that would otherwise result in nonterminating calls to getTypeParameters().

Objects of this class are created by LazyParameterBound.substitute(Substitution) when the substitution would replace a type variable with another type variable.
