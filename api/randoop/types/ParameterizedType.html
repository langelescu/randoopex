public abstract class ParameterizedType extends ClassOrInterfaceType
Represents a parameterized type. A parameterized type is a type C<T1,...,Tk> where C<F1,...,Fk> is a generic class instantiated by a substitution [Fi:=Ti], and Ti is a subtype of the upper bound Bi of the type parameter Fi.
