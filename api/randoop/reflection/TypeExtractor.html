Class TypeExtractor

java.lang.Object
randoop.reflection.DefaultClassVisitor
randoop.reflection.TypeExtractor
All Implemented Interfaces:
ClassVisitor
class TypeExtractor extends DefaultClassVisitor
TypeExtractor is a ClassVisitor that extracts both the class type, and concrete types that are used in a class as either a parameter, a return type, or a field type.
